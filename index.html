1. 개인정보의 수집 및 이용 목적

BaroMap은 다음의 목적을 위해 개인정보를 처리합니다:

  • 위치 기반 서비스 제공: 사용자 주변의 편의시설(화장실, 주차장, 약국, 병원, 편의점, 맛집)을 검색하고 안내하기 위한 목적으로만 사용됩니다.
  • 앱 기능 개선: Firebase Analytics를 통해 익명화된 사용 패턴을 분석하여 서비스 품질을 향상시킵니다.

2. 수집하는 개인정보 항목

2.1 필수 수집 정보

  • 위치 정보(GPS): 사용자의 현재 위치를 기반으로 주변 편의시설을 검색합니다.
  • 기기 정보: 앱 버전, 기기 모델, OS 버전 등 (서비스 최적화 및 오류 분석 목적)

2.2 선택 수집 정보

  • 앱 사용 통계: 검색한 카테고리, 사용한 기능 등 (Firebase Analytics를 통해 익명으로 수집)

3. 개인정보의 보유 및 이용 기간

  • 위치 정보: 앱 실행 중에만 메모리에 임시 저장되며, 서버에 전송되거나 저장되지 않습니다.
  • 사용 통계: Firebase Analytics에서 자동으로 14개월 후 삭제됩니다.
  • 앱 삭제 시: 모든 로컬 데이터가 즉시 삭제됩니다.

4. 개인정보의 제3자 제공

BaroMap은 사용자의 개인정보를 외부에 판매하거나 제공하지 않습니다. 단, 다음의 서비스를 이용합니다:

  • Kakao Local API: 주변 장소 검색 (위치 좌표만 전송, 개인 식별 정보 없음)
  • Google Gemini API: AI 기반 장소 설명 생성 (장소명과 카테고리만 전송, 사용자 정보 없음)
  • Firebase (Google): 익명화된 사용 통계 분석 및 앱 성능 모니터링

1. Purpose of Collection and Use of Personal Information

BaroMap processes personal information for the following purposes:

  • Location-based Services: To search and guide nearby facilities (toilets, parking, pharmacies, hospitals, convenience stores, restaurants) around the user.
  • App Improvement: Analyze anonymized usage patterns via Firebase Analytics to enhance service quality.

2. Personal Information Items Collected

2.1 Required Information

  • Location Information (GPS): Used to search for nearby facilities based on the user's current location.
  • Device Information: App version, device model, OS version, etc. (for service optimization and error analysis)

2.2 Optional Information

  • App Usage Statistics: Searched categories, used features, etc. (collected anonymously via Firebase Analytics)

3. Retention and Use Period of Personal Information

  • Location Information: Temporarily stored in memory only while the app is running. It is NOT transmitted to or stored on servers.
  • Usage Statistics: Automatically deleted after 14 months in Firebase Analytics.
  • Upon App Deletion: All local data is immediately deleted.

4. Provision of Personal Information to Third Parties

BaroMap does not sell or provide user personal information to external parties. However, the following services are used:

  • Kakao Local API: Search for nearby places (only location coordinates sent, no personal identification information)
  • Google Gemini API: Generate AI-based place descriptions (only place name and category sent, no user information)
  • Firebase (Google): Anonymized usage statistics analysis and app performance monitoring

5. Consignment of Personal Information Processing

BaroMap outsources personal information processing tasks as follows:

Consignee Consigned Task
Google LLC (Firebase) App usage statistics analysis, performance monitoring
Kakao Corp. Place search service provision
Google LLC (Gemini API) AI-based place description generation

6. Rights and Obligations of Data Subjects and How to Exercise Them

Users can exercise the following rights:

  • Refuse Location Data Collection: iOS Settings > BaroMap > Location > "Never" (note: core app features will be unavailable)
  • Data Deletion: All local data is automatically deleted when the app is deleted
  • Inquiries and Requests: Contact us anytime via the contact information below
